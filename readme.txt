Use this plugin to format your Power BI Codes (DAX and M)

For M code (PowerQuery)
[mcode theme=dark lineWidth=90 width=600px ]let a = 1 in a[/mcode]

For DAX code (PowerPivot)
[daxcode width=600px region=EU]EVALUATE myTable[/daxcode]

All shortcode attributes are optional.

For API documentations see:
DAX: https://www.sqlbi.com/blog/marco/2014/02/24/how-to-pass-a-dax-query-to-dax-formatter/
M: https://www.powerqueryformatter.com/api

The formatted results for unchanged code are cached for 30 days via WordPress transients.

Github Repo: https://github.com/mogular/wp-powerbicodeformatter
